# VoyageDiscovery 🌍
### Immersive AI-Powered Travel Engine for the 2026 Google Hackathon

VoyageDiscovery is a next-generation travel orchestrator designed to transform how we discover and experience the world. Built on a foundation of **Sensory-First Discovery**, the platform moves beyond traditional search filters to prioritize the "vibe," emotional resonance, and environmental impact of your journey.

![VoyageDiscovery Logo](https://fonts.gstatic.com/s/i/short-term/release/googlesymbols/auto_awesome/default/48px.svg)

## 🚀 Key Features

- **Sensory Discovery (Vibe Search)**: Forget "Beach" or "City." Search by "Oceanic Resonance," "Midnight Pulse," or "Ethereal Calm." Our engine uses AI to match destinations to your desired sensory output.
- **Global Pulse**: A real-time social discovery layer that integrates community stories, live sentiment analysis, and quick-booking capabilities.
- **Split-Stay Orchestrator**: AI-optimized multi-destination itineraries that balance luxury, logistics, and local immersion seamlessly.
- **Secure by Design**: Built with a privacy-first mindset, utilizing a secure local storage orchestration layer to protect user PII.
- **Sustainability Pulse**: Future-proofed integration with **Google Earth Engine** to provide real-time verified environmental data for every destination.

## 🛠️ Tech Stack

- **Core**: React 18, JavaScript (ES6+)
- **Styling**: Tailwind CSS (Premium Material Design 3 Palette)
- **Icons & Fonts**: Google Fonts (Plus Jakarta Sans, Inter), Material Symbols
- **Hosting**: Google Cloud Storage (Static Website Hosting)
- **Deployment**: Google Cloud Build pipeline
- **Data**: Centralized Discovery Mock Repository (Scalable for Vertex AI/Firestore integration)

## 🎨 Design Philosophy: "Guided Exploration"
VoyageDiscovery balances the reliability of institutional travel platforms with the boutique sensibility of an editorial magazine.
- **Corporate Modern**: Clean, white-heavy foundation to allow photography to shine.
- **Airy Layouts**: Utilizes premium spacing tokens (`xxxl`, `xxxxl`) to reduce cognitive load and create a high-end, breathy feel.
- **Accessibility**: 100% WCAG 2.1 AA compliant, featuring semantic HTML5, aria-menubar roles, and high-contrast color pairings.
